# Healthcare Analytics Projects

## 📌 Overview

This folder contains healthcare analytics projects focused on patient outcomes, survival prediction, and clinical data analysis using advanced statistical and machine learning techniques.

---

## 🏥 Projects

| Project | Tools | Description |
|---------|-------|-------------|
| [Colorectal Cancer Survival Analytics](Colorectal-Cancer-Survival/) | MySQL · Power BI · Python | Analyzed 89,945 patient records to identify key factors influencing survival. Built an interactive 11-page Power BI dashboard. |
| [Heart Failure Prediction](Heart-Failure-Prediction/) | Python · Scikit-learn · Jupyter | Implemented 6 machine learning models to predict heart failure mortality. Achieved 83.33% accuracy with Tuned Random Forest (AUC-ROC: 0.914). |

---

## 📊 Key Skills Demonstrated

| Skill | How It Was Applied |
|-------|-------------------|
| **SQL** | Data extraction, cleaning, and analysis of 89,945 patient records |
| **Power BI** | Interactive 11-page dashboard development with KPIs and slicers |
| **Machine Learning** | XGBoost, Random Forest, SVM, Logistic Regression, KNN, Gradient Boosting |
| **Feature Engineering** | Feature scaling, hyperparameter tuning (GridSearchCV) |
| **Statistical Analysis** | Survival analysis, regression, EDA (correlation, boxplots, pairplots) |
| **Data Visualization** | Clear storytelling with charts, maps, and dashboards |
| **Healthcare Domain** | Patient records, clinical data, survival outcomes, mortality prediction |

---

## 🎯 Business Impact

- **Colorectal Cancer Study** – Identified that the **Under 40 age group** is the largest cohort (~25,850 patients), challenging conventional assumptions about the disease.
- **Heart Failure Prediction** – Identified **ejection fraction** and **serum creatinine** as the strongest clinical predictors, enabling early intervention.
- **Actionable Insights** – Produced clear recommendations for screening programs and treatment optimization.
