# Telecom Analytics Projects

## 📌 Overview

This folder contains telecom analytics projects focused on customer behavior, churn prediction, and revenue optimization using advanced statistical and machine learning techniques.

---

## 📡 Projects

### Customer Churn Prediction
- **Tools:** Python · Stan · Bayesian Modeling
- **Description:** Developed a Bayesian hierarchical model to predict customer churn for telecom operators
- **Key Achievement:** AUC-ROC of 0.941, outperforming Random Forest (0.912) and Logistic Regression (0.885)
- **Folder:** [Customer-Churn-Prediction/](Customer-Churn-Prediction/)

---

## 📊 Key Skills Demonstrated

| Skill | How It Was Applied |
|-------|-------------------|
| **Bayesian Modeling** | Hamiltonian Monte Carlo sampling in Stan |
| **Predictive Analytics** | Customer churn prediction |
| **Model Comparison** | AUC-ROC evaluation against multiple models |
| **Uncertainty Quantification** | Posterior distributions and 95% credible intervals |
| **Python** | Data processing, modeling, and visualization |

---

## 🎯 Business Impact

- Identified key drivers of customer churn
- Provided insights for retention strategies
- Quantified uncertainty in predictions for better decision-making
